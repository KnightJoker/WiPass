## 需要

- XCode 9.0 +
- Target of iOS 9 +
- CocoaPods 1.1.0 +

## 如何运行

- 进入到预定的项目路径
`cd /path`

- 克隆项目
`git clone git@github.com:KnightJoker/WiPassword.git`

- 使用 CocoaPods 来管理一些必要的三方库
```
$   cd WiPassword/
$   pod install         
// 如果 pod 更新速度过慢，您可以使用这条命令 `pod install --verbose --no-repo-update`
```

- 然后打开项目中的 `.xcworkspace`  文件，就可以运行项目了
